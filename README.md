# カラーコード出力ツール

ブラウザ上で色を選んで、  
**HEX / RGB / HSL を一発で確認＆コピー＆CSVダウンロード**できるツールです。

## 機能

- カラーピッカーで色を選択
- 選択した色を HEX / RGB / HSL で表示
- 各形式ごとにコピーアイコンでワンクリックコピー
- 選択中の色コードを大きく表示
- ダークモード / ライトモード切り替え
- 現在の色コードを CSV 形式でダウンロード

## ファイル構成

- `index.html` … 本体の構造
- `style.css` … 見た目（デザイン）
- `script.js` … 動き（ロジック）

## 使い方

1. ブラウザで `index.html` を開く
2. 「Picker」で色を選ぶ
3. 必要な形式（HEX / RGB / HSL）のコピーアイコンを押してコピー
4. 「CSVでDL」ボタンで現在の色コードを `color_codes.csv` として保存

## メモ

- クリップボードコピーには `navigator.clipboard` を使用
- テーマ切り替えは CSS カスタムプロパティ（`--bg` など）で実装
- 小学生クリエイターによるガチ作品
